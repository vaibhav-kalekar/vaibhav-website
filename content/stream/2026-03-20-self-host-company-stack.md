I self hosted Campfire and Writebook for internal chat and documentation on my own server. Seamless set up and even added a sub domain for overall access.

Just starting with Fizzy and liking it so far and will self host it as well.

One of the main reasons to self host some of these services is the availability of software like ONCE which makes it easy to set up and maintain.

Early days but I am liking the set up.

Great opinionated software used to be norm before all the enshitified subscription based slop filled the market.

This is just one way the tech world seems to be normalising.
